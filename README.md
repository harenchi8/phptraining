# ✅ PHP + Laravel 学習ロードマップ（週末3時間×16週 / JS経験者向け）

JavaScriptの基礎知識がある前提で、PHP特有の文法、DB連携、LaravelフレームワークによるWebアプリ開発を段階的に習得していく学習プランです。
